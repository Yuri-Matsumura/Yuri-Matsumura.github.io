# Yuri Matsumura — GitHub Pages

This is a plain HTML/CSS website designed for GitHub Pages.

## Publish/update

1. Put `index.html`, `assets/`, and `jp/` at the root of the `yuri-matsumura.github.io` repository.
2. Keep existing files such as `cv_yuri_matsumura.pdf` and `jmp_00draft.pdf` at the repository root.
3. Commit and push to the publishing branch (usually `main`).

## Edit papers and links

All English text and paper URLs are in `index.html`.

- To change a title's destination, edit only the `href="..."` inside the title's `<a>` tag.
- To add a paper, duplicate an existing top-level `<li> ... </li>` in the relevant section.
- The layout is intentionally minimal: headings, standard bullets, and direct title links.
